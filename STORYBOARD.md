# SHELLY URBAN — סטוריבורד אתר אימרסיבי

**סגנון:** Active Theory × Obys Agency · גישה **היברידית** (WebGL לסצנות מרכזיות + CSS/3D ל-fallback)
**עמוד-שדרה:** מסע מצלמה אחד רציף לאורך מחזור-החיים של בניין — *מקרקע גולמית ועד מסירת מפתח*.
**Back-end:** Base44 מחזיק DATA + לוגיקה (ישויות פרויקטים, טופס לידים/CRM). שכבת ה-WebGL היא הקליפה.
**Mood:** בהיר ואוורירי (פלטה קיימת `#f1efea` + ירוק) — הר אביטל הוא האובייקט המרחף המרכזי.
**פרויקט הדגל:** **הר אביטל** — הדמיות אמיתיות ב-`assets/har-avital/` (day · night · side).

---

## פרויקט הדגל — הר אביטל

מגדל מגורים מודרני (~9–10 קומות) עם שפה אדריכלית חזקה ששזורה בכל הסצנות:
- **חזית פחם אלגנטית** + **עמוד עץ/נחושת חם** אנכי (נקודת המוקד).
- **מרפסות מרחפות** + **לוחות-רצפה קונזוליים** (קווים אופקיים).
- **כתר גג** שלוח עם פרגולה (חתימה).
- **תאורת LED לינארית** שמדגישה כל קומה — בלילה הבניין זוהר.

**הדמיות → סצנות:**
| קובץ | סצנה | שימוש |
| --- | --- | --- |
| `har-avital-day.jpeg` | Hero / Projects (יום) | האובייקט המרחף המרכזי, מצב יום |
| `har-avital-night.jpeg` | Hero hover / Contact | זוהר LED — micro-moment דרמטי, "מסירת מפתח" בערב |
| `har-avital-side.jpeg` | Domains / Lightbox | זווית שנייה, מקור למונוליתים |

**פלטה שחולצה (אקסנטים, מעל הבהיר):** עץ/נחושת `#a8623c` · פחם `#2c2a30` · בטון `#b6b1a8` · זוהר LED `#f3d6a0` · ירוק צמחייה `#5f7d3c` (קרוב לירוק המותג, נשאר אקסנט).

> ⚙️ ההדמיות נשמרות ברזולוציית מקור (5K). גרסאות web-optimized (WebP/AVIF, מספר גדלים) ייוצרו בשלב הבנייה.

> מסמך זה הוא ה-blueprint לפני כתיבת קוד. כל סצנה כוללת: מצלמה · מעבר · טיפוגרפיה · אינטראקציה · שכבה טכנית (WebGL/CSS) · נכסים · חיבור Base44 · fallback מובייל.

## 0 · מערכות גלובליות (רצות לאורך כל האתר)

| מערכת | תיאור | שכבה |
| --- | --- | --- |
| **Smooth scroll** | אינרציה רכה (Lenis-style), הגלילה מניעה את ציר הזמן של המצלמה | JS |
| **Custom cursor** | נקודה ירוקה מגנטית; משתנה לטבעת מעל אלמנטים אינטראקטיביים | CSS/JS |
| **Grain overlay** | שכבת גרעין עדינה קבועה (חתימת Obys) | CSS/canvas |
| **Sound toggle** | hum רקע + ticks עדינים באירועים; כבוי כברירת מחדל, toggle בפינה | JS (WebAudio) |
| **Scene manager** | ניהול מצב הסצנה הפעילה, lazy-load של נכסים כבדים | JS |
| **Reduced-motion** | `prefers-reduced-motion` → מכבה WebGL, עובר ל-CSS-fallback סטטי-עדין | JS |
| **RTL** | כל הטיפוגרפיה והפריסה עברית RTL, הגופנים Frank Ruhl Libre + Heebo | CSS |

**פלטה (מהאתר הקיים):** רקע `#f1efea`, דיו `#17151a`, ירוק `#5f9c36` / bright `#84bf52` / deep `#447327`.
**מצלמה:** מסלול יחיד מוגדר מראש (spline); הגלילה = פוזיציה על המסלול. אין "קפיצות סקשן" — מעברים רציפים.

## 00 · Preloader — "התעוררות השרטוט"

- **מצלמה:** סטטית, מבט-על על שרטוט שטוח.
- **רצף:** מסך כהה + גרעין → קו ירוק משרטט רשת blueprint → הלוגו `SHELLY URBAN` מתאסף מחלקיקים → מונה אחוזים (`0 / 100`) בטיפוגרפיה מונו עם letter-spacing רחב.
- **מעבר יציאה:** ב-100% השרטוט השטוח "מתרומם" ל-3D, המצלמה נסוגה ומגלה מגרש קרקע → זורם ישר ל-Hero.
- **שכבה:** WebGL (חלקיקים) + CSS fallback (קו SVG שמצויר + fade).
- **נכסים:** `assets/s-mark-green.png`, shader חלקיקים, sample tick.
- **Base44:** —
- **מובייל:** גרסה קצרה — לוגו + בר טעינה, בלי חלקיקים כבדים.

## 01 · Hero — "הקרקע"

- **מצלמה:** אורביט איטי סביב מגרש תלת-ממדי מרחף בערפל וולומטרי.
- **רכיבים:** מגרש 3D, **הר אביטל מרחף מעליו** (הדמיית `day`), אור ירוק מותגי.
- **micro-moment:** ב-hover/שהייה, קווי ה-LED של הבניין נדלקים קומה-קומה (מעבר עדין ל-`night`) ואז חוזרים — רמז לדרמה בלי לשבור את ה-mood הבהיר.
- **טיפוגרפיה קינטית:** **"כאן ייבנה הבית הבא שלכם."** — האותיות נחשפות בעיוות WebGL (text reveal). תת-כותרת: תקציר 3 התחומים.
- **אינטראקציה:** parallax לפי עכבר (tilt), סמן מגנטי, scroll-cue "גלילה".
- **שכבה:** WebGL (fog + plane + render plane). Fallback: הדמיית הר אביטל מרחפת עם bob ב-CSS.
- **נכסים:** `har-avital/har-avital-day.jpeg` (+`night` ל-hover), fog/ground shader, HDR-lite env.
- **Base44:** טקסט hero ניתן לעריכה כ-content entity (אופציונלי).
- **מובייל:** מגרש סטטי + ההדמיות הקיימות, בלי orbit.

## 03 · Domains — "שלושה מונוליתים"

- **מצלמה:** עוצרת מול שלושה עמודי-אבן תלת-ממדיים מרחפים.
- **תוכן:** א׳ התחדשות עירונית · ב׳ מחיר למשתכן · ג׳ בנייה למגורים.
- **המראה:** המונוליתים הם **שלוש פאות של הר אביטל** — אותה שפה אדריכלית (לוחות קונזוליים, עמוד העץ, כתר הגג). כל אחד מייצג תחום.
- **אינטראקציה:** hover מדליק את **קווי ה-LED** של המונוליט (כמו בהדמיית הלילה), מסובב אותו וחושף הטקסט — כמו בחירת "שלב" במשחק.
- **שכבה:** WebGL (3 meshes + emissive LED + lighting). Fallback: שלוש כרטיסיות `.dom` הקיימות עם hover-line.
- **נכסים:** `har-avital/har-avital-side.jpeg` כטקסטורה, shader emissive ל-LED.
- **Base44:** שלושת התחומים כ-entity (כותרת + טקסט), ניתן לעריכה.
- **מובייל:** שלוש כרטיסיות מוערמות, hover→tap.

## 04 · Projects — "העיר" ⭐ (הסצנה המרכזית)

- **מצלמה:** מסע אופקי מעל מפת-עיר תלת-ממדית מסוגננת; כל פרויקט = בניין שטסים אליו.
- **אינטראקציה:** hover על בניין מרים אותו + מציג תווית; קליק → ההדמיה "מתממשת" במעבר WebGL לתוך לייטבוקס עם הפרטים.
- **פרויקטים:** הלילך 7/5/3 (יבנה), הציונות 10 · סיטי פארק (אשדוד), קרן היסוד · מתחם הצעירים (אשדוד), העצמאות 28–32 (פינוי-בינוי).
- **שכבה:** WebGL (instanced buildings + image-transition shader). Fallback: ה-`.grid` + `.feat` + לייטבוקס הקיימים.
- **נכסים:** כל ה-`proj-*.jpg`, shader מעבר תמונה (RGB-displace).
- **Base44:** ⚡ **כאן החיבור העיקרי** — רשימת הפרויקטים נטענת מ-Base44 entity (`title, location, status, image, description`). הוספת/עריכת פרויקט ב-Base44 → מתעדכן אוטומטית בסצנה.
- **מובייל:** גריד כרטיסים אנכי + לייטבוקס; בלי מפת-עיר.

## 05 · Execution — "מהשטח / הבנייה עולה"

- **מצלמה:** עולה לאורך בניין שנבנה תוך כדי גלילה.
- **רצף:** מעבר wireframe → solid; עגורנים ופיגומים; הבניין מתרומם קומה-קומה (assembly לפי scroll progress).
- **אינטראקציה:** התמונות האמיתיות ממופות כמשטחים; hover → grain + RGB-shift (חתימת Obys).
- **שכבה:** WebGL (build animation). Fallback: ה-`.exec-grid` הקיים עם reveal.
- **נכסים:** `exec-site.jpg`, `exec-crane.jpg`, `exec-done.jpg`, wireframe shader.
- **Base44:** תמונות "מהשטח" כ-gallery entity (אופציונלי).
- **מובייל:** שלוש תמונות reveal, בלי build-animation.

## 08 · Contact — "מסירת המפתח"

- **מצלמה:** מתייצבת מול בניין מוגמר; אייקון מפתח; סגירת המסע.

## ארכיטקטורת Base44 (החיבור "בצד השני")

```
┌─────────────────────────────────────────┐
│  Front-end אימרסיבי (HTML/JS/WebGL)      │  ← הקליפה / החוויה
│  Three.js · shaders · Lenis · cursor     │
└───────────────┬─────────────────────────┘
                │  fetch / SDK
┌───────────────▼─────────────────────────┐
│  Base44 (DATA + לוגיקה)                  │  ← המוח
│  Entities:                               │
│   • Project (title, location, status,    │
│              image, description)          │
│   • Domain  (key, title, text)            │
│   • Stat    (number, label)               │
│   • Partner (logo, link)                  │
│   • Lead    (name, phone, project, msg)   │
│   • SiteContent (hero/contact texts)      │
└──────────────────────────────────────────┘
```

- **קריאה:** סצנות 03/04/06/07/08 נטענות מ-entities → עדכון תוכן בלי נגיעה בקוד.
- **כתיבה:** טופס Contact → entity `Lead` (CRM/לידים).
- **Embed:** קובץ ה-front-end משולב כ-custom code/embed בתוך אפליקציית Base44 (ראה `SETUP.md`).

## חבילה טכנית מוצעת (היברידי)

- **3D:** Three.js (+ postprocessing קל). טעינה עצלה — נטען רק כשהסצנה הראשונה מוכנה.
- **Scroll:** Lenis (או מימוש פנימי קל כמו ב-`index.html` הקיים).
- **Build:** ללא build חובה — ESM מ-CDN כדי שיתאים ל-embed של Base44. אופציה ל-bundle אם צריך.
- **ביצועים:** budget של ~60fps דסקטופ; DPR מוגבל; instancing לבניינים; texture compression; degrade אוטומטי למובייל.
- **Fallback:** כל סצנה כוללת מקבילה ב-CSS/3D — האתר עובד מלא גם בלי WebGL.
